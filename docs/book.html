Style

"style" : {
  "txtcol" : "#fff", 
  "bgcol": "#000" , 
  "btncol" : "#fff", 
  "btnbg" : "#333", 
  "btnotl" : "2px solid #666", 
  "btntxtN" : "&#8592;", 
  "btntxtP" : "&#8594;" 
}

txtcol: The color of text on every page
bgcol: The background color of every page
btncol: The color of text on buttons
btnbg: The background color of buttons
btnoutl: The outline of buttons
btntxtN: The text of the next page button
btntxtP: The text of the previous page button
